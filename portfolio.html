Portfolio

Gwitter

Gwitter is a clone of Twitter that I am working on as a side project to practice & improve my skills in React, CSS design, and NodeJS. This self project aims to replicate the inner workings of Twitter and use distributed systems design to make a decentralized social media network in the future.

Japanese Hiragana/Katakana App

This application is a personal project of mine to help me learn and memorize Hiragana and Katakana's alphabet and aid me in my journey to becoming better at Japanese. Currently, I am improving this project to add more game mechanics to the game, such as a leaderboard and more visual effects.

Aurora Automotive

Aurora Automotive, which specialized in fleet management, needed a marketing surveying form filled by potential customers. A cloud Firebase instance stores the survey data, React was used for the front-end implementation, with Material UI components. This application used functional programming principles for clean, maintainable application design.

Command Line Music Player

A music player inspired by command-line application interfaces and connected to a youtube streaming service enables the user to listen to music. This project is developed using Python and ncurses.

Conway's Game of Life

Conway's game of life was something I wanted to create, and I implemented the simulation using Python and the processing library for the graphic rendering. The simulation works on four simple rules which simulate the mechanics of the game.

Merkle Tree using Stack

I had a fascination for distributed systems and decentralized system design. Therefore I stumble upon the Merkle tree, which is used to verify the blockchain in cryptocurrencies. Therefore I decided to implement one using Python.
