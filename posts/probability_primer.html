Basic Set Theory

A set is a collection of objects, which are the elements of the set.

A set \(S\) can be defined as \( S = \{x_1, x_2, ...., x_n\}\)

When \( x \in S \), \(x\) belongs to the set \(S\)

When \(x \notin S\), \(x\) doesn't belong to the set \(S\)

If every element in a set \(T\) is also an element of set \(S\), then we can say that set \(T\) is a subset of set \(S\), denoted by \( T \subset S \)

E.g. If set \(T = \{ x_1, x_2, x_3 \} \) and set \( S = \{x_1, x_2, x_3, .... x_n \}\),
then we can state that \( T \subset S \)

We can also define a universal set \(\Omega \), to be a set that has all the possible values that are in given context, where every possible set follows the rule, \( S \subset \Omega \)



A set which has elements that don't belong to the set, is known as the complement of that set, defined by \[ S^c = \{ x \in \Omega | x \notin S \} \]



Union of two sets is defined by the following operation, \[ S \cup T = \{ x | (x \in S) or (x \in T) \} \]



Intersection of two sets is defined by the following operation, \[ S \sqcup T = \{ x | (x \in S) and (x \in T) \} \]



Extending this two operations, we can have union/intersection for multiple sets like so, \[ \bigcup_{i=1}^{\infty} S_{i} \] \[ \bigcap_{i=1}^{\infty} S_{i} \]



Two sets or more are disjoint if their intersection is empty. \( S \cap T = \emptyset \)

Here is the generalized rule, \[ \bigcap_{i=1}^{n} S_{i} = \emptyset \]

Two or more sets are a partion of a bigger set \(V\), if \( S \cup T = V \), or more generally, \[ \bigcup_{i=1}^{n} S_{i} = V \]



Here are several set properties, used to build relationships between sets, and simplify set algebra \[ S \cup T = T \cup S \] \[ S \cap (T \cup V ) = (S \cup T) \cap (S \cup V)\] \[ S \cup (T \cap V ) = (S \cap T) \cup (S \cap V)\] \[ (S^{c})^{c} = S \] \[ S \cap S^c = \emptyset \] \[ S \cap \Omega = S \] \[ S \cup \Omega = \Omega \]



Here we prove the Demorgan's law, which is a law used throughout probability theory

First lets prove the union law, of the Demorgan's law

\[ (S \cup V)^c = S^c \cap V^c \] \[ x \in (S \cup V)^c \] \[ x \notin S \text{ and } x \notin V \] \[ x \in S^c \text{ and } x \in V^c \] \[ \text{Therefore, } x \in (S^c \cap V^c )\]




Second lets prove the interesection law of the Demorgan's law

\[ (S \cap V)^c = S^c \cup V^c \] \[ x \in (S \cap V)^c \] \[ x \notin S \text{ or } x \notin V \] \[ x \in S^c \text{ or } x \in V^c\] \[ \text{Therefore, } x \in (S^c \cup V^c )\]




Probabilistic Models

A mathematical description of an uncertain situation

Elements of a probabilistic model include the sample space \( \Omega \), which outlines all the possible outcomes of an experiment

The probability law assigns a nonnegative number to a set A of possible outcomes. \[ A = \{x_1, x_2, x_3 \} \] \[ P(x_1) \geq 0 \] \[ P(x_2) \geq 0 \] \[ P(x_3) \geq 0 \] \[ P(x_1) + P(x_2) + P(x_3) = 1 \]

The probabilistic model involves an underlying process called the experiment, which produces exactly one out of several possible outcomes. Different elements of the sample space should be distinct and mutually exclusive, so that when the experiment is carried out there is an unique outcome. \[P(A) \geq 0, \text{for every event } A \] \[P(A \cup B) = P(A) + P(B) \text{ iff } A \cap B = \emptyset \]

\[ \text{ if the sample space is discrete & } A_1 \text{ to } A_n \text{ are disjoint, then } P(\bigcup_{i}^{n} A_i) = P(A_1 \cup A_2 \cup A_3 .... \cup A_n) = P(A_1) + P(A_2) + ... + P(A_n) = \sum_{i=1}^{n} P(A_i) \] \[P(\Omega) = 1 \] \[\text{If } A \subset B \text{ then } P(A) \leq P(B) \] \[P(A \cup B) = P(A) + P(B) - P(A \cap B) \] \[P(A \cup B) \leq P(A) + P(B) \] \[P(A \cup B) = P(A) + P(B) \text{ iff } A \cap B = \emptyset\] \[P(A \cup B) = P(A) + P(A^c \cap B) \] \[\text{Generally, for a sequence of events A, } P(\bigcup_i^n A_i) \leq \sum_i^n P(A_i)\]

Conditional Probability

Conditional probability is about finding probabilities of events, given prior information about the environment.

\[ \text{If } P(B) \text{ > 0, then } P(A|B) = \frac{P(A \cap B)}{P(B)} \]

\[\text{If } P(B) \text{ > 0, then } P(\Omega|B) = \frac{P(\Omega \cap B)}{P(B)}\]

\[ \text{If any two events are disjoint, } P(A_1 \cup A_2 | B) = P(A_1 | B) + P(A_2 | B) \]

\[\text{If any given number of n events, which are disjoint, } P(\bigcup_{i=1}^{n} A_i | B ) = P(A_1 | B ) + ... + P(A_n | B ) \]

\[\text{Generally, for any given event A,B and C : } P(A \cup C | B) \leq P(A|B) + P(C|B)\]

\[\text{Generally, } P(\bigcup_{i=1}^{n} A_i | B ) \leq P(A_1 | B ) + ... + P(A_n | B ) \]

Multiplication Rule

\[ P(A \cap B) = P(B)P(A|B) = P(B) . \frac{P(A \cap B)}{P(B)} = P(A \cap B)\]

\[ \text{Generally, } P(\bigcap_{i=1}^{n} A_i) = P(A_1) . P(A_2 | A_1) . P(A_3 | A_2 \cap A_1) ... P(A_n | \bigcap_{i=1}^{n-1} A_i)\]

Total Probability Theorem

If \( A_1, A_2 , ... , A_n\) are disjoint events, and \( A_1 \cup A_2 \cup A_3 \cup ... \cup A_n = \Omega \), then assuming that \( P(A_i) \gt 0 \text{ } \forall \text{ } i \), if any event \( B \) occurs, then \[ P(B) = P(A_1)P(A_1 \cap B) + P(A_2)P(A_2 \cap B) + ... + P(A_n)P(A_n \cap B) = \sum_{i=1}^{n} P(A_i)P(A_i \cap B)\]

Deriving the Baye's Rule

If \( A_1, A_2 , ... , A_n\) are disjoint events, and \( A_1 \cup A_2 \cup A_3 \cup ... \cup A_n = \Omega \), then assuming that \( P(A_i) \gt 0 \text{ } \forall \text{ } i \), if any event \( B \) occurs, then

\[P(A_i | B) = \frac{P(B \cap A_i)}{P(B)}\]

\[P(B|A_i) = \frac{P(B \cap A_i)}{P(A_i)} \text{, therefore }\] \[P(B \cap A_i) = P(A_i) . P(B|A_i) \]

\[ \text{By subsitution, we get the Baye's rule, like so, } \] \[ P(A_i | B) = \frac{P(A_i) . P(B|A_i)}{P(B)} \]

\[ \text{Since } P(B) = \sum_{i=1}^{n} P(A_i)P(A_i \cap B) \text{ by the total probabiliy theromn, we can generalize further, }\] \[ P(A_i | B) = \frac{P(A_i) . P(B|A_i)}{\sum_{i=1}^{n} P(A_i)P(A_i \cap B)} \]
