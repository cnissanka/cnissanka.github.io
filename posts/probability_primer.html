A Probability Theory Cheat Sheet

This is a cheat sheet in all the probability theory fundamentals of basic probability theory. We are going to quickly go over basic set theory, define a probability of an event, define what conditional probability means, independence of events & conditional independence, finally proving Bayes theroem.

Basic Set Theory

A set is a collection of objects, which are the elements of the set.

A set \(S\) can be defined as \( S = \{x_1, x_2, ...., x_n\}\)

When \( x \in S \), \(x\) belongs to the set \(S\)

When \(x \notin S\), \(x\) doesn't belong to the set \(S\)

If every element in a set \(T\) is also an element of set \(S\), then we can say that set \(T\) is a subset of set \(S\), denoted by \( T \subset S \)

E.g. If set \(T = \{ x_1, x_2, x_3 \} \) and set \( S = \{x_1, x_2, x_3, .... x_n \}\),
then we can state that \( T \subset S \)

We can also define a universal set \(\Omega \), to be a set that has all the possible values that are in given context, where every possible set follows the rule, \( S \subset \Omega \)



A set which has elements that don't belong to the set, is known as the complement of that set, defined by \[ S^c = \{ x \in \Omega | x \notin S \} \]



Union of two sets is defined by the following operation, \[ S \cup T = \{ x | (x \in S) or (x \in T) \} \]



Intersection of two sets is defined by the following operation, \[ S \sqcup T = \{ x | (x \in S) and (x \in T) \} \]



Extending this two operations, we can have union/intersection for multiple sets like so, \[ \bigcup_{i=1}^{\infty} S_{i} \] \[ \bigcap_{i=1}^{\infty} S_{i} \]



Two sets or more are disjoint if their intersection is empty. \( S \cap T = \emptyset \)

Here is the generalized rule, \[ \bigcap_{i=1}^{n} S_{i} = \emptyset \]

Two or more sets are a partion of a bigger set \(V\), if \( S \cup T = V \), or more generally, \[ \bigcup_{i=1}^{n} S_{i} = V \]



Here are several set properties, used to build relationships between sets, and simplify set algebra \[ S \cup T = T \cup S \] \[ S \cap (T \cup V ) = (S \cup T) \cap (S \cup V)\] \[ S \cup (T \cap V ) = (S \cap T) \cup (S \cap V)\] \[ (S^{c})^{c} = S \] \[ S \cap S^c = \emptyset \] \[ S \cap \Omega = S \] \[ S \cup \Omega = \Omega \]



Here we prove the Demorgan's law, which is a law used throughout probability theory

First lets prove the union law, of the Demorgan's law

\[ (S \cup V)^c = S^c \cap V^c \] \[ x \in (S \cup V)^c \] \[ x \notin S \] and \[ x \notin V \] \[ x \in S^c \] and \[ x \in V^c \] \[ x \in (S^c \cap V^c )\]




Second lets prove the interesection law of the Demorgan's law

\[ (S \cap V)^c = S^c \cup V^c \] \[ x \in (S \cap V)^c \] \[ x \notin S \] or \[ x \notin V \] \[ x \in S^c \] or \[ x \in V^c \] \[ x \in (S^c \cup V^c )\]
