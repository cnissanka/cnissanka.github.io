Basic Probability Theory - Independence

Independence of events can be viewed as, if there are two events A & B, given that B happened, does not change your beliefs about A. The fact that B happened, doesn't affect the probability of A. It remains the same. Under such a scenario, we can state that A and B are independent events from one another.

\[ P(A|B) = P(A) \] \[ P(A|B) = \frac{P(A \cap B)}{P(B)} = P(A)\] \[ \text{Therefore, } P(A)P(B) = P(A \cap B)\] \[ \text{If } P(A \cap B) = P(A)P(B) \text{ then we can state that A & B are independent events}\] \[\text{ Disjoint events are under most scenarios not independent.}\] \[\text{If } P(A \cap B) = \emptyset \text{ then } P(A \cap B) = 0 \text{, but} P(A) \gt 0 \text{ & } P(B) \gt 0 \text{, therefore } P(A \cap B) \ne P(A)P(B)\]

\[\text{Independence in multiple events}\] \[ P(A \cap B | C) = \frac{P(A \cap B \cap C)}{P(C)} = P(A|C)P(B|C) \] \[ \text{We can write the intersection as follows, } P(A \cap B \cap C) = P(C)P(A | C)P(B | C \cap B)\] \[ \text{By subsitution we get } P(A \cap B | C ) = \frac{P(C)P(A | C)P(B | C \cap B)}{P(C)} = P(A|C)P(B|C) \] \[ \text{Afterwards } P(A | C)P(B | C \cap B) = P(A|C)P(B|C)\] \[ P(B | C \cap B) = P(B|C) \text{ If events A and B are independent under the condition of C}\] \[ \text{Generally, } P( \bigcap_{ i \in s} A_i) = \prod_{i \in s} P(A_i | C)) \text{ for any given subset i of s where } s = \{1, 2, ..., n\}\]

P(A_n | B) \] \[ \text{If any event } i \gt 0 \text{ is independent then, } P(A_{i+1} | B) = P(A_{i+1} | (\bigcap_{j=0}^i A_{j+1}) \cap B) \text{ such that } (i+1) \in N\]
